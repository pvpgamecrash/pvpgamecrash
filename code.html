How To Make A Circle In Javascript

Copy and Paste To see How it Works!

//Create jsGraphics object
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.beginPath();
ctx.arc(100, 75, 50, 0, 2 * Math.PI);
ctx.stroke();
