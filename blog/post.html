My new game!: BREAKOUT!

Read more to find out...

Posted by Talon Bragg on September 29, 2016

This is one of my very first games. I got this game form MDN (Mozilla Developer Network), It wasn't all that great at first, so I decided to make it better. I changed the variables so that It would be harder. I also added a new function so on the last level you do not start the game intantainiously.

The main object of the game is to break the bricks until they are all gone. When you do this, you advance to the next level. If your ball hits the ground to many times, it says, "Game Over", and you lose, you have 3 lives on the first level, you have 2 lives on the second level, and you have one life on the third level.

Please play Breakout.
